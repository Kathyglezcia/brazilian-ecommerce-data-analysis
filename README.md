# Brazilian E-Commerce Data Analysis

This project explores customer behavior, product demand, payment preferences, and delivery performance using the Brazilian Olist E-Commerce dataset.

The objective is to extract business insights that help understand purchasing patterns, logistics efficiency, and market demand.

---

## Dataset

The dataset used in this project comes from the **Olist Brazilian E-Commerce dataset**, which contains information about:

- Orders
- Customers
- Products
- Payments
- Reviews
- Delivery information

The data represents real e-commerce transactions in Brazil.

---

## Tools & Technologies

- Python
- Pandas

---

## Analysis Performed

The project includes several business analyses:

### Product Category Demand
Identified which product categories generate the highest number of orders.

### Orders by City
Analyzed which cities contribute the most to order volume.

### Payment Method Distribution
Examined how customers prefer to pay for their purchases.

### Delivery Time Analysis
Measured how long it takes for orders to be delivered to customers.

### Monthly Order Trend
Analyzed order growth over time to identify trends in platform activity.

---

## Key Findings

- Home, electronics, and furniture categories generate the highest order volume.
- São Paulo dominates the platform's order activity.
- Credit cards are the most common payment method.
- Most deliveries occur within a relatively short timeframe.
- The platform shows signs of growth based on monthly order trends.
